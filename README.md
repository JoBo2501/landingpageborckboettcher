# Landingpage Borck Böttcher Legal

## Beschreibung
Moderne, professionelle Landingpage für die Rechtsanwaltskanzlei Borck Böttcher Legal.

## Technologie-Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design

## Struktur
```
/
├── index.html          # Haupt-HTML-Datei
├── css/
│   └── styles.css      # Styling
├── js/
│   └── main.js         # JavaScript-Funktionalität
├── assets/
│   ├── images/         # Bilder und Grafiken
│   └── fonts/          # Schriftarten
└── README.md           # Projektdokumentation
```

## Features
- Responsive Design für alle Endgeräte
- Moderne UI/UX
- Kontaktformular
- Leistungsübersicht
- Team-Vorstellung
- Impressum & Datenschutz

## Entwicklung
Dieses Projekt befindet sich in aktiver Entwicklung.

## Landingpage lokal ansehen
1. Repository aktualisieren oder klonen:
   ```bash
   git clone <repo-url>
   cd landingpageborckboettcher
   git pull
   ```
2. Platzieren Sie das Kanzleifoto als `assets/images/kanzlei.jpg`, damit der Hero-Bereich und das Profilbild korrekt angezeigt werden.
3. Öffnen Sie `index.html` direkt im Browser oder starten Sie einen lokalen Webserver, z. B.:
   ```bash
   python -m http.server 8000
   ```
   und rufen Sie anschließend `http://localhost:8000` auf.
