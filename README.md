# Landingpage Borck Böttcher Legal

## Beschreibung
Moderne, professionelle Landingpage für die Rechtsanwaltskanzlei Borck Böttcher Legal.

## Technologie-Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design

## Struktur
```
/
├── index.html          # Haupt-HTML-Datei
├── css/
│   └── styles.css      # Styling
├── js/
│   └── main.js         # JavaScript-Funktionalität
├── assets/
│   ├── images/         # Bilder und Grafiken
│   └── fonts/          # Schriftarten
└── README.md           # Projektdokumentation
```

## Features
- Responsive Design für alle Endgeräte
- Moderne UI/UX
- Kontaktformular
- Leistungsübersicht
- Team-Vorstellung
- Impressum & Datenschutz
